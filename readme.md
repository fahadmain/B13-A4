1. What is the difference between getElementById, getElementsByClassName, and querySelector / querySelectorAll?
Ans. getElementById: Selects one element by its id. Returns a single element object.
    getElementsByClassName: Selects elements by class name.
    querySelector: Selects the first matching element.
    querySelectorAll: Selects all matching elements.

3. What is Event Bubbling? And how does it work?
Ans. Event bubble up from the target element to its ancestors.

4. What is Event Delegation in JavaScript? Why is it useful?
Ans. A technique where a parent handles events of child elements using bubbling. It is useful for dynamic content.

5. What is the difference between preventDefault() and stopPropagation() methods?
Ans. The core difference is that preventDefault()  stops the browser's default behavior for an event, while stopPropagation() stops the event from traveling through the DOM to parent or child elements.
